# School IT Help Desk System

## What this project does
This system lets students to submit regular IT issues, view them and update them.

## Features
- Add a new ticket
- View all tickets
- Update ticket status

## Tools used
- Python
- SQLite

## How to run
Enter this on terminal:
python3 app.py

Then use the menu and enter the number of your choice.

## Database
Uses a local SQLite database file (tickets.db).
